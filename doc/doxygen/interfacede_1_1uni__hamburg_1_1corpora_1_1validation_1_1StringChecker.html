Report check (String data)
 

Detailed Description

A HZSK validator that can check data from a string. Most validators should start with this. The data is typically file contents as string, but not necessarily. The validator should create a statistics report while checking.

This would not be used anymore if all the Checker implement the CorpusFunction and all the Checks the Check Interface.
