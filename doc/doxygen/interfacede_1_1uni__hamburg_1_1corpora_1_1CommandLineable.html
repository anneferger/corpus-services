Report doMain (String[] args)
 

Detailed Description

A command-line interface for error-checking corpora files. This ensures that a check has an entry point that can be called after construction. This is basically a non-static main, but it can be called from other contexts too, so it provides a neat entry point. A typical implementation goes like so:

public StatisticsReport doMain(String[] args) {
  settings.handleCommandLine(args, ...);
  for (File f : settings.getInputFiles()) {
    checker.check(f);
  }
  public static void main(String[] args) {
    FooBarChecker checker = new FooBarChecker();
    checker.doMain(args);
  }
}
